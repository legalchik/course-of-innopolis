# Основы ООП. Урок 1.

1. Реализовать класс Child, имеющего атрибуты имя, фамилия, класс в котором он учится. При создании класса вы можете использовать getter, setter или же dunder методы get, set.

2. Создать базовый класс и два дочерних класса с типами персонажа. Классы должны иметь атрибуты имя, кол-во жизней, броня. Реализовать метод атаки, в котором персонажи атакуют друг друга пока жизни какого то из них, не будут равны или меньше нуля.
