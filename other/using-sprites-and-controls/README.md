# Использование спрайтов и управление

1.Создать поверхность. Найти в интернете изображение персонажа(или использовать моего), добавить его на дисплей.
Реализовать управление по стрелкам при помощи изменение координат и присваивания этих координат спрайту при помощи функции set_position().

2.Добавить ещё один спрайт(монетки или любой другой) для будущего взаимодействия с персонажем и вывести их на экран.
